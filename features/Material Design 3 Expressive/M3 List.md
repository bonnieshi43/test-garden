# M3 Expressive 样式适配清单

## 一、M3 Expressive 风格简介

Material Design 3 Expressive 是适合大屏/数据分析类产品的设计风格，核心特点：

- **圆角更大**：组件圆角 12-28px（比常规 M3 大 30-50%）
- **间距更宽松**：基准间距 12-16px，增加呼吸感
- **色彩更鲜明**：支持动态颜色，对比度更高
- **动效更丰富**：强调缓动曲线，时长 200-400ms

**适用场景**：报表分析、大屏展示、创意工具
